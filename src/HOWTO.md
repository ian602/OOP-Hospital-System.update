# Hospital Management System

## Project Description
This project is a simple Hospital Management System developed using Java and Object-Oriented Programming principles.

The system allows users to enter patient information and display it. The program demonstrates OOP concepts such as classes, objects, constructors, and encapsulation.

## Features
- Add patient details
- Display patient information
- Dynamic user input using Scanner
- Exception handling using try, catch, and finally

## Exceptions Implemented

1. InputMismatchException
   Handles situations where the user enters invalid input such as letters instead of numbers.

2. General Exception Handling
   Prevents the program from crashing when unexpected errors occur.

3. Finally Block
   Ensures resources such as Scanner are properly closed.

## How to Run the Program

1. Open the project in IntelliJ IDEA.
2. Run the Main class.
3. Enter patient details when prompted.
